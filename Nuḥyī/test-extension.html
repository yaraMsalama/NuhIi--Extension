Installation Instructions

  1. Open Chrome/Edge and go to chrome://extensions/
  2. Enable "Developer mode" (toggle in top right)
  3. Click "Load unpacked" and select the Nuḥyī extension folder
  4. The extension should appear in your extensions list
  5. Click the extension icon to open the popup

Expected Features

  • Quran Tab: Random Quranic verses with Arabic text and translation
  • Azan Tab: Prayer times with notifications and sound
  • Adhkar Tab: Morning/evening/sleep Adhkar with categories
  • Reminders Tab: Custom reminders and Salah on Prophet notifications
  • Settings: Dark mode, location, Azan audio, reminder customization

Known Issues & Fixes

  • Prayer Times Error: Fixed by using HTTPS API calls
